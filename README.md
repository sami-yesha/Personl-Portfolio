# Samuel Yeshambel - Backend Developer Portfolio

A modern, high-performance portfolio website built to showcase backend engineering expertise and full-stack capabilities. Designed with a focus on visual excellence, smooth interactions, and accessibility.

- **Modern & Premium Design**: 
    - Glassmorphism effects

    - "Warm Stone & Emerald" color palette
- **Interactive Elements**:
    - **Dark/Light Mode**: Fully dynamic theming with a custom animated toggle.
    - **Micro-interactions**: Hover effects on cards, buttons, and links.
    - **Responsive Mobile Menu**: Smooth overlay with staggered animations.
- **Tech Stack**:
    - **Framework**: Next.js 16 (App Router)
    - **Styling**: Tailwind CSS v4 (using `@theme` and OKLCH colors)
    - **Language**: TypeScript
    - **Animations**: Framer Motion
    - **Icons**: Lucide React
    - **Theming**: next-themes
